How to install?
===============================================
!!!!!!!!!!!!!  Attention  !!!!!!!!!!!!

The bin. files were compiled on Ubuntu 24.04 x86_64 linux.

You should double check the OS verions before connecting vpn by softether vpn client.

1. copy bin files at usr/local/bin/* to /usr/local/bin
   
   cp -arfp usr/local/bin/* /usr/local/bin/
   
3. copy vpnse to /usr/local/bin
   
   cp -arfp vpnse /usr/local/bin/
   

You can run vpnse now.
===============================================
Usage: vpnse conncect/start/status/disconnect/stop/add/del account_namev

       vpnse conn van2869
       vpnse disconn van2869
       vpnse start van2869
       vpnse stop van2869
       vpnse status van2869
       vpnse add van2869
       vpnse del van2869
